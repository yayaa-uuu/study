#### 进制转换

| 进制      | 数码  | 基数 | 位权             |
|---------|-----|----|----------------| 
| 十进制(D)  | 0-9 | 10 | 10<sup>K</sup> |
| 二进制(B)  | 0-1 | 2  | 2<sup>K</sup>  |
| 十六进制(H) | 0-F | 16 | 16<sup>K</sup> |

#### 按权展开法

D 10100.001 = 1 * 10<sup>4</sup> + 1 * 10<sup>2</sup> + 1 * 10<sup>-3</sup>

R<sup>k</sup> => R为基数，k与小数点的距离有关

#### 短除法

十进制转化为R进制使用短除法

概念

1. 商为0截止
2. 余数从下往上记录

|       | 余数 | 
|-------|----|
| 2\|94 | 0  | 
| 2\|47 | 1  |
| 2\|23 | 1  |
| 2\|11 | 1  |
| 2\|5  | 1  |
| 2\|2  | 0  |
| 2\|1  | 1  |
| 0     |    |

即94的二进制表示方式为1011110  
使用按权展开法验证 1 * 2<sup>6</sup> + 1 * 2<sup>4</sup> + 1 * 2<sup>3</sup> + 1 * 2<sup>2</sup> + 1 * 2
=64+16+8+4+2
