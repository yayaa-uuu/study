#### 缓存穿透，击穿，雪崩
1. 穿透
查询不存在的数据。
解决方案
非法参数校验。
布隆过滤器。
缓存这些不存在的数据。

2. 击穿
key突然过期，请求都查询数据库
解决方案
互斥锁
逻辑过期
永不过期


3. 雪崩
大量的key过期，缓存不可用
解决方案
key设置不同的过期时间
缓存保证高可用，哨兵模式，集群
服务降级和熔断
缓存预热
