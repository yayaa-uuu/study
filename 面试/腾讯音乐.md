15. 冒泡排序算法

```java
for(int i=0;i<a.length-1;i++)
for(int j=0;j<a.length-1-i;j++){
    if(a[j]>a[j+1]){
        int temp=a[j];
        a[j]=a[j+1];
        a[j+1]=a[j];
    }
}

插入排序
for(int i=0;i<a.length-1;i++)
for(int j=i;j>0&&a[j]<a[j-1];j--){
        int temp=a[j];
        a[j]=a[j-1];
        a[j-1]=a[j];
}
```
