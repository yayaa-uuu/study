#### String StringBuffer 和 StringBuilder 的区别是什么?

|  | 可变性 | 线程安全性 | 使用 |
|--|--|--|--|
| String | × | √ | 少量数据 |
| StringBuilder | √ | × | 字符串大量数据(单线程) |
| StringBuffer | √ | √ | 字符串大量数据(多线程) |

String 的不可变性   底层是final修饰的数组，且没有公开对数组的修改
String  str = "a";      字符串常量池中直接构建对象 
String  str = new String("a");     堆上一个，常量池一个
String.intern();      放到常量池
